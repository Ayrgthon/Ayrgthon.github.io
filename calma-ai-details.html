Project Overview

CALMA is an interactive web application designed to be a digital sanctuary and proactive tool for daily self-care and emotional health management. The application proposes caring for a capybara as a reflection of personal self-care, integrating an intelligent emotional journal and wellness chatbot.

Core Features

  • Intelligent Emotional Journal: Write your thoughts and let our AI analyze the sentiment to color your emotional calendar
  • Adaptive Daily Challenges: Based on your journal entries, the application suggests personalized tasks to help you manage your emotions
  • Capybara Sensei: A conversational chatbot (powered by an LLM) that offers you a safe space to talk, listen, and receive advice based on wellness techniques
  • Subtle Gamification: The concept of caring for your pet (the Capybara) through your own self-care (completing challenges) creates a positive motivation cycle
  • Clean and Reactive Interface: Built with modern web technologies for a smooth and relaxing user experience

System Demonstration

System Architecture

CALMA's architecture is designed to be modular and efficient, allowing smooth interaction between the frontend, a mock API for data persistence, and an external AI service for sentiment analysis.

  • Frontend: React with Vite for a fast and powerful interface
  • Mock Backend: Node.js & Express.js for simulated RESTful API
  • Database: JSON file as local NoSQL database
  • AI: Llama 3B Instruct for sentiment analysis and task generation

Technical Challenges

Several complex technical challenges were addressed during development, including ethical gamification, accurate sentiment analysis, data privacy, and smooth user experience.

  • Ethical Gamification: Balancing user motivation with respect for mental health
  • Accurate Sentiment Analysis: Ensuring the AI model correctly interprets user emotions without bias
  • Data Privacy: Protecting sensitive user emotional information while maintaining functionality
  • Smooth User Experience: Creating an interface that is both engaging and not overwhelming for users in vulnerable emotional states

Technical Roadmap

Planned technical improvements focus on expanding the application's capabilities, improving user experience, and extending system integration with additional data sources.

  • Wearable Integration to obtain biometric data that enriches emotional analysis
  • Wellness Marketplace with more content and personalized tools
  • User Community to create safe spaces for mutual support
  • AI Model Improvements for more accurate sentiment analysis
