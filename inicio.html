Predicción de Ausentismo con Datos Limitados

Esta guía interactiva explora las metodologías avanzadas para predecir la ocurrencia y duración del ausentismo laboral a partir de un conjunto de datos muy pequeño (N=100) y complejo. Descubra cómo abordar los desafíos de datos escasos y variables categóricas jerárquicas con técnicas robustas.

Iniciar Exploración

El Desafío: Calidad sobre Cantidad

Modelar con solo 100 datos presenta dos obstáculos principales que requieren un enfoque especializado.

1. Datos Limitados (N=100)

Con una muestra pequeña, el riesgo de que el modelo "memorice" los datos en lugar de aprender patrones generales (**sobreajuste**) es extremadamente alto.

  • ✓
    Modelos Simples y Regularizados: Se prefieren modelos menos complejos que son inherentemente más robustos.
  • ✓
    Validación Robusta (LOOCV): Se utiliza la Validación Cruzada Dejando-Uno-Fuera para maximizar los datos de entrenamiento.

2. Variables Categóricas Jerárquicas

Variables como "Diagnóstico General -> Específico" o "País -> Región -> Ciudad" contienen una estructura jerárquica valiosa.

Ejemplo: Codificación Simple (One-Hot)

`País-Región-Ciudad`

`COL-ATL-BQA` → `[1,0,0, 1,0,0, 1,0,0]` (9 cols)

Problema: ¡Explosión de dimensiones!

Solución de Codificación Inteligente

La clave es transformar las categorías en números significativos sin crear demasiadas características.

Codificación Jerárquica Explicada

1. Principio Básico: En lugar de codificar cada nivel independientemente, se crea una codificación que considera la estructura jerárquica completa.

2. Ponderación por Nivel: Se asignan pesos diferentes a cada nivel de la jerarquía:

  • Nivel Ciudad (más específico): 50% del peso
  • Nivel Región: 30% del peso
  • Nivel País (más general): 20% del peso

3. Fórmula de Codificación:

Valor_Final = (Media_Ciudad × 0.5) + (Media_Region × 0.3) + (Media_Pais × 0.2)
Flujo de Codificación:

Ejemplo ilustrativo: Codificación de una ubicación jerárquica

  • Ciudad (BQA): 2 de 3 casos de ausentismo → Media = 0.667
  • Región (ATL): 3 de 4 casos de ausentismo → Media = 0.750
  • País (COL): 3 de 5 casos de ausentismo → Media = 0.600

Flujo de Modelado en Dos Etapas

El problema se divide en dos preguntas secuenciales, cada una con su propio modelo especializado.

ETAPA 1

Clasificación

¿Tendrá el empleado ausentismo (Sí/No)?

ETAPA 2

Regresión

Si la respuesta es "Sí", ¿cuántos días?

Análisis: ¿Qué Variables Influyen Más?

Más allá de predecir, queremos entender los factores clave.

Importancia por Permutación

Se "desordena" aleatoriamente una variable y se mide cuánto empeora el modelo.

Coeficientes del Modelo

En modelos simples como la Regresión Logística, la magnitud del coeficiente de una variable indica su peso en la predicción final. L1 (Lasso) puede incluso reducir a cero los coeficientes de variables no importantes.

Análisis de Clustering

Busca agrupar a los individuos en "perfiles de riesgo" (ej. bajo, medio, alto) basados en sus características, para identificar patrones de comportamiento comunes.

Ejemplo Visual: Importancia de Variables

Conclusión y Hoja de Ruta

Modelar con datos escasos es un ejercicio de rigor y pragmatismo.

Pasos Estratégicos Recomendados:

  • 1
    Codificación Robusta: Implementar Codificación por Objetivo Regularizada.
  • 2
    Clasificación Primero: Usar Regresión Logística con regularización (L1/L2) para el modelo de Sí/No. Evaluar con métricas como F1 y AUC-PR.
  • 3
    Regresión para los Positivos: Para el subconjunto con ausentismo, usar un Modelo de Obstáculo (Hurdle) con componente Binomial Negativo para predecir los días.
  • 4
    Interpretar y Analizar: Utilizar técnicas de importancia de variables para entender los factores clave, no solo para predecir.
  • !
    Gestionar Expectativas: Con N=100, el objetivo es construir el modelo más fiable posible y generar hipótesis, no necesariamente predicciones perfectas.
